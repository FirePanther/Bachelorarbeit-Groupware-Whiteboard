Class: Selection

Selection

new Selection(el, yopt, wopt, hopt)

Creates a selection element to show which element has the focus. The element gets an animated dashed border. It is possible to set the coordinates (x, y, width, height) or to give an element as parameter so the coordinates can be determined automatically.
Parameters:
Name Type Attributes Description
el Object/int The x coordinate or a jQuery element.
y int <optional>
 The y coordinate. Ignored, if first parameter is an element.
w int <optional>
 The width. Ignored, if first parameter is an element.
h int <optional>
 The height. Ignored, if first parameter is an element.
