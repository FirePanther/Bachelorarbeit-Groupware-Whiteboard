createTextfield(userIdopt, idopt, focusopt, valopt, leftopt, topopt, widthopt, heightopt, coloropt, opacityopt, fsizeopt, familyopt, redrawopt)

Creates a new textfield containing a textarea and a label. The parameters will be set to the textfield. If this is not a redraw this method broadcasts the textfield with its parameters. The textfield will be added into the textfields Object created in the constructor.
Parameters:
Name Type Attributes Default Description
userId string <optional>
 0 The user who created this textfield.
id string <optional>
 text_{userId}_{serverTime} The id for this textfield.
focus boolean <optional>
 false If the textfield should get the focus.
val string <optional>
 "" The value for this textfield.
left int <optional>
 {min:click.x1,click.x2} The left coordinate.
top int <optional>
 {min:click.y1,click.y2} The top coordinate.
width int <optional>
 {max(click.x1,click.x2)-left} The width for this textfield.
height int <optional>
 {max(click.y1,click.y2)-top} The height for this textfield.
color string <optional>
 {selected color} The color for the textarea and label.
opacity float <optional>
 {selected opacity} The opacity for the textarea and label.
fsize int <optional>
 {setting font size} The font size for the textarea and label.
family string <optional>
 {setting font family} The font family for the textarea and label.
redraw boolean <optional>
 false If this is a redraw.
